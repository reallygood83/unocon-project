## 전체 화면 구성

### 공통 레이아웃

모든 페이지는 다음과 같은 공통 레이아웃을 공유합니다:

```
+----------------------------------+
|            헤더/내비            |
+----------------------------------+
|                                  |
|                                  |
|            메인 콘텐츠           |
|                                  |
|                                  |
+----------------------------------+
|              푸터               |
+----------------------------------+
```

#### 헤더/내비게이션
- 로고: 왼쪽 상단에 유니콘 로고
- 메인 메뉴: 주요 서비스로 연결되는 네비게이션 링크
  - 홈
  - 읽어보며 통일 알아보기
  - 영상으로 통일 알아보기
  - UN참전국 감사편지 보내기
- 반응형: 모바일에서는 햄버거 메뉴로 변환

#### 푸터
- 저작권 정보
- 통일부 통일교육원 링크
- 개인정보 처리방침
